# Gundam 3D E-commerce Website

This project is part of the graduation thesis titled:

**"Developing a Web Application for Selling Gundam Models with Integrated 3D Visualization"**

## 🧾 Description

This is a full-stack web application built for selling Gundam model kits. The website includes a wide range of features such as:

- User authentication and authorization (Admin, Staff, Customer)
- Product management
- Cart and checkout system
- Order tracking
- Coupon and discount system
- Notification system (for promotions, orders, and system messages)
- Review and feedback system
- 3D product preview (using integrated 3D model viewer)
- Chatbot integration using Dialogflow

The goal of the project is to improve the online shopping experience for Gundam enthusiasts by providing interactive and immersive product visualizations.

## 🚀 Tech Stack

### Frontend:
- React.js + Tailwind CSS
- Zustand for state management
- React Router
- Axios
- Framer Motion (animations)

### Backend:
- Node.js + Express
- MongoDB (Mongoose ODM)
- JWT for authentication
- Multer (for image upload)
- Cloudinary (image hosting)
- Dialogflow (chatbot integration)

## 📦 Features

| Module | Features |
|--------|----------|
| 🛒 Product | Browse, search, 3D preview, wishlist |
| 👤 User | Sign up, login, profile management |
| 🔐 Roles | Admin, Staff, Customer role-based access |
| 📦 Orders | Place, view, update status |
| 💬 Reviews | Post, reply, moderate reviews |
| 📢 Notifications | Real-time user notifications |
| 💬 Chatbot | AI assistant via Dialogflow |
| 🎨 Admin Panel | Manage products, users, orders, coupons, policies |
| 🧩 Settings | Store settings, maintenance mode |

## 🖼️ 3D Integration

3D models are rendered using:
- [Three.js](https://threejs.org/)
- [React Three Fiber](https://github.com/pmndrs/react-three-fiber)

This allows customers to preview Gundam models in real-time with zoom, rotate, and pan functionalities.

## 🤖 Chatbot Integration

The site includes a custom chatbot UI integrated with **Dialogflow Essentials**, capable of handling customer inquiries about:

- Product details
- Order status
- Support policies
- Promotions and more
